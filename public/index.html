Flow

快速接入流程

  1. 1创建商户录入商户资料，配置结算与费率策略。
  2. 2配置通道启用支付插件，维护通道密钥和支付方式。
  3. 3获取密钥为商户生成 API 凭证，完成签名联调。
  4. 4发起支付按文档创建订单，跳转收银台完成支付。
  5. 5接收通知处理异步通知，失败任务由系统自动重试。
